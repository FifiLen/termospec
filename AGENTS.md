## Core rule

Do not invent product structure, content, colors, routes, or architecture from scratch.  
If something is not defined, add it as an open question in `context/progress-tracker.md` before implementing.

## Project character

This is a professional B2B website redesign for Termospec. Treat it as an industrial / engineering / technical company site, not a generic SaaS landing page and not a generic HVAC template.
